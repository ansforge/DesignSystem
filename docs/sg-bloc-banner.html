Mise en place

Ce bloc doit être collé visuellement au <header>. Pour ce faire :

  • charger les CSS suivantes :
    • CSS du Système de Design
    • CSS globale pour les blocs (1,9Ko - CSS)
    • CSS du bloc Bannière (6,1Ko - CSS)
  • le placer en première position de la balise <main>,
  • modifier la variable CSS --main-margin-top, elle doit avoir pour valeur 0 : 
    
                            :root {
                                --main-margin-top: 0;
                            }
