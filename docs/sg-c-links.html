Composants

Liens

Un lien hypertexte crée un chemin vers une page web, un fichier, une adresse mail ou encore un emplacement (id) dans la page courante.

Liens dans le corps de texte

Pour les liens qui renvoient vers une ressource externe (target="_blank"), le JS ajoute une icône ainsi qu'un title pour l'accessibilité.

La contextualisation des liens est importante : il convient donc d'éviter les ancres de lien du type "En savoir plus" ou "Lien".
