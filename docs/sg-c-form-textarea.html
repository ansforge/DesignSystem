Composants - Formulaire

Zone de texte

Représentée par l'élément textarea, il s'agit d'une zone permettant la saisie d'un texte sur plusieurs lignes.

Le champ « zone de texte »

Par défaut

Afin de garantir une certaine homogénéité entre les navigateurs, il peut être utile de définir les attributs rows et cols pour fixer une taille unique à l'élément.

Les différents états

Tout comme les champs de saisie, plusieurs états sont possibles.

Champ obligatoire

Une fois l'attribut required ajouté au textarea, ce dernier doit comporter une value non nulle pour que le formulaire puisse être validé.

Désactivé

L'ajout de l'attribut booléen disabled empêche la saisie de texte et la prise de focus de l'élément.

Message

Peut être utile pour spécifier le type d'informations requises pour la prise en compte de la demande.

Description helper text

Message d’erreur

Si une erreur est détectée dans la valeur saisie, la classe is-invalid peut être ajoutée au div.form-group ainsi qu'au textarea. Le message d'erreur doit être explicite et si possible prévoir des suggestions de correction.

Error helper text
